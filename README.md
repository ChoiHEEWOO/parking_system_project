parking system, RFID, server, DB


현재 진행 상황
================================================



-RC-522 모듈 2개 사용.
------------------------------------------------
CH0: 입구 리더기

CH1: 출구 리더기

입구 리더기에서 카드 입력 시, ESP8266을 통해 서버로 데이터가 전송이 된다. 또한 DB에 등록된 회원의 경우 UID가 버퍼에 저장되어 
현재 주차장에 존재하는 차량으로서 기록이 된다.

출구 리더기의 경우, 태그 시 소리만 나고 기록되어 있던 UID정보를 버퍼에서 지운다.
